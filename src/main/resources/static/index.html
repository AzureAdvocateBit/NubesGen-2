NubesGen - the cloud infrastructure generator

Default configuration

Download as a ZIP file:
curl https://nubesgen.azurewebsites.net/nubesgen.zip -o nubesgen.zip

Download and unzip automatically:
curl https://nubesgen.azurewebsites.net/nubesgen.zip | jar xv

Using parameters

Download as a ZIP file:
curl https://nubesgen.azurewebsites.net/nubesgen.zip -d location=westeurope -d applicationName=myapplication -o nubesgen.zip

Download and unzip automatically:
curl https://nubesgen.azurewebsites.net/nubesgen.zip -d location=westeurope -d applicationName=myapplication | jar xv

Available parameters

Paramater name Default value Description
location eastus The Azure location where all resources in this example should be created
applicationName sampleNubesApplication The name of your application
